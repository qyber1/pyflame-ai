# PyFlame AI

Инструмент предназначен для сбора статистики скорости работы Python кода, для дальнейшнего анализа и рефаторинга.

Быстрый старт:

1. ```pip install git+https://github.com/qyber1/pyflame-ai.git```
2. ```pyflame-ai config init```
2. ```pyflame-ai run --path <script-path>```
3. ```pyflame-ai open-report```

Обзор команд:  
| Команда | Флаг | Назначение |  
|---------|------|------------|  
| run | `-p/ --path`   | Указать путь до файла, который будет анализироваться |  
| | `-o/ --ouput-filename`   | Название файла для вывода py-spy. По умолчанию `profile_cli.txt` |  
| | `-s/ --samples`    | Количество сэмплов для анализа |
| | `--api-key`        | Обязательный параметр - ключ для доступа к модели |
| | `--dry-run`        | Тестовый режим |
| config | | Инициализация и обновление файла конфигурации |
| open-report | `-f/ --filename` | Путь до файла для вывода отчета в консоль |
| | `-r/ --raw` | Вывод отчета в исходном виде |
